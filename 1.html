Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications.
Web browsers reseive HTML documents form a web sever of from local storage and render them into multimedia web pages. HTML describes the structure of a web page semantically and originally included cues for the appearance o fthe document.

스타일부분은 CSS 다른언어인데 약속되어 사용된다. HTML elements are the building blocks of HTML pages. With HTML constructs, images and other objects, such as interactive forms, may be embedded into the rendered page. It provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
